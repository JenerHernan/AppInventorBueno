VARIABLES

(HECHO)
Las variables son contenedores donde guardar valores y poder repetir
estos en multiples partes del código y poder modificarlo todo a 
la vez de manera rápida y sencilla. 

//Explicar todo esto en una sola imagen (HECHO)
Podemos llamar a nuestras variables de casi cualquier manera, aun así hay
ciertas reglas que debemos seguir. En python hay ciertas 
reservadas para la sintaxis del lenguaje. Por ejemplo no podemos llamar
a una variable "for", ya que esta es la palabra reservada para los 
bucles for. Tampoco podemos comenzar el nombre por un numero.



//Combinar este y el anterior (HECHO)
Tenemos que tener en cuenta que el compilador distingue entre mayúsculas y
minúsculas. La variable numero y Numero se considerarian como dos nombres
diferentes.





FUNCIONES

//HECHO
Las funciones permiten repetir código usando una etiqueta. Esto nos ahorra 
lineas de código duplicadas y hace que el código sea más comprensible.

//HECHO
Las funciones pueden, o no, tener varios parámetros de entrada. Las funciones
trabajarán con los parámetros, para procesarlos y devolver un parámetro de salida.

//HECHO
Un concepto que debemos tener en cuenta cuando usamos funciones es el del ámbito 
de una variable. El ámbito es donde se puede usar una variable en el código. 


Por ejemplo, una variable declarada en una función, únicamente podrá ser usada dentro 
de esa función.  Es decir, es de ámbito local a la función. Si queremos usar una misma 
variable en varias funciones, tenemos que declararla en el ámbito global, fuera de cualquier función.
